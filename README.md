# Landing page - Felipe Soares | ES11 Capital

Landing page pessoal de Felipe Soares, socio da ES11 Capital. O projeto utiliza Next.js 16 e React 19.

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra `http://localhost:3000` no navegador.

## Validacao

```bash
npm run build
```

## Publicacao na Vercel

Importe este repositorio na Vercel usando a branch `main`. O projeto e detectado como Next.js; mantenha o comando de build padrao (`npm run build`) e nao defina uma pasta de saida manualmente.

## Estrutura principal

- `app/`: landing page, artigos e metadados.
- `public/`: fotos, logo e imagem de compartilhamento.
- `output/pdf/`: documento de pendencias e justificativas.
